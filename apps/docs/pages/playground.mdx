# Playground

The interactive playground allows you to experiment with Websmith Kit components, customize themes, and generate code snippets.

## Features

- **Component Preview**: See components with different props and variants
- **Theme Switching**: Switch between preset themes or create custom ones
- **Code Generation**: Copy ready-to-use code snippets
- **Real-time Updates**: See changes instantly

## Using the Playground

1. Select a component from the sidebar
2. Adjust props using the controls
3. Switch themes to see how it looks
4. Copy the generated code
