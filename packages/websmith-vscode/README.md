# Websmith Kit VS Code Extension

Enhances your developer experience with Websmith Kit:

- Component tree explorer with quick insert and create
- Design token snippets and exports
- Theme creation, validation, and preview helpers
- IntelliSense completion and hover docs for components and tokens

## Commands

- Websmith: Create Component
- Websmith: Create Theme
- Websmith: Export Design Tokens
- Websmith: Validate Theme Configuration
- Websmith: Generate TypeScript Declarations
- Websmith: Open in Playground
- Websmith: Open Storybook
- Websmith: Insert Component
- Websmith: Insert Design Token
- Websmith: Preview Theme

## Snippets

- Components: `ws-button`, `ws-card`, `ws-dialog`, `ws-avatar`, `ws-badge`, `ws-label`, `ws-input`, `ws-select`, `ws-import`
- Tokens: `ws-color`, `ws-spacing`, `ws-font-size`, `ws-shadow`, `ws-border-radius`, `ws-var`, `ws-token-css`, `ws-token-scss`, `ws-token-ts`

## Development

- Build: `npm run compile`
- Test locally: Open this folder in VS Code and run the "Launch Extension" debug task.
