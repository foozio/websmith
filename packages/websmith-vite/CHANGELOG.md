# @websmith/vite

## 1.0.0

### Major Changes

- Initial release of Vite plugin for Websmith design tokens
- Automatic CSS custom property generation
- Hot module replacement for token changes
- Support for multiple export formats (CSS, JSON, TypeScript)
- Configurable output paths and prefixes
- Integration with Vite's build system
- Virtual module support for direct token imports

### Features

- Token generation from Websmith design system
- File watching with automatic regeneration
- Production build optimization
- TypeScript type generation
- Custom token configuration support
- Theme selection and customization

### Plugin API

- `websmith()` plugin function with comprehensive options
- Virtual module `websmith:tokens` for direct imports
- Integration with Vite's watcher and HMR system

### Documentation

- Complete README with usage examples
- Configuration options documentation
- Integration guide for various project types
